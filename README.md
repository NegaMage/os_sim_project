https://github.com/NegaMage/os_sim_project/tree/1.0

## Requirements

### Ubuntu

TKinter is installable with 
`
sudo apt install python-tk
`

### Windows

Follow the detailed instructions at https://tkdocs.com/tutorial/install.html#installwin 

## Running the program

Go into the directory and run
`
python3 base.py
`
on Ubuntu or 
`
python base.py
`
on Windows.

A GUI display built on tkinter will have the rest of the steps for each individual program
